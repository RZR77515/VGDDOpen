Upload Updates

With this page you can upload new/updated files to the HTTP server which is running on the GUIBoard™.

This allows you to change/update web server functionality without having to reflash your board.

Web server root is set to ~webpages~, but you can upload contents in any other folder, too: if the folder doesn't exist, it will be created.
